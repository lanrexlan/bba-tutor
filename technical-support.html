🛠️ Fixes

Troubleshooting Flows

Follow the steps in order. Most issues resolve within 3–5 minutes.

📹 Camera/Mic Not Detected

  1. Close other apps/tabs using camera/mic.
  2. Browser → Site settings → Allow camera & microphone.
  3. Select the correct input/output devices.
  4. Unplug/replug USB headset; try another port.
  5. Restart browser. If still failing, restart computer.

🔊 Students Can't Hear/See Me

  1. Use a headset; turn laptop speakers down.
  2. Reduce video resolution to 360p; disable incoming video if needed.
  3. Move closer to router; pause downloads/streams.
  4. Switch to 5 GHz Wi‑Fi or a wired connection.

🖥️ Screen Share Isn't Working

  1. Share a single tab/window for lower bandwidth.
  2. Close sensitive content; mute notifications.
  3. Enable "Share system audio" only when needed.
  4. Update browser to the latest version.

🐢 Platform Feels Slow

  1. Close unused tabs/programs, especially video apps.
  2. Disable virtual background; use plain wall.
  3. Clear browser cache and restart.
  4. Try a different device if lag persists.
